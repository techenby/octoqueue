# Where to Get Printer URL

There are multiple ways for you to connect your printer to OctoQueue. The easiest way is to us OctoEverywhere but using ngrok or setting up a public url yourself are also perfectly viable options.

## OctoEverywhere

- Login or create a free account
- Set up your printers with OctoEverywhere
- Create a shared connection
- Name it OctoQueue
- Select the printer
- Make sure that "Expires" is unchecked
- Click "Create"
- Copy url and paste into OctoQueue
